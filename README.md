# Valet-Parking-System

## 프로젝트 소개
발렛파킹 로봇을 이용한 무인주차시스템을 설계하고 구현한 프로젝트입니다.
ArUco 마커 인식, 메카넘 휠 제어, 센서 기반 거리 측정, 리프팅 동작을 통해 차량을 자동으로 탐색하고 주차하는 시스템입니다.

## 개발 목적
좁은 주차 공간에서 사람의 개입을 줄이고, 자동화된 주차 프로세스를 구현하는 것을 목표로 하였습니다.

## 시스템 구성
- Jetson Nano: 영상 처리 및 마커 인식
- STM32: 모터 및 센서 제어
- 메카넘 휠: 전방향 이동
- 초음파/ToF 센서: 거리 측정
- 리니어 액추에이터: 로봇 길이 조절
- 서보 모터: 차량 리프팅 장치
- BLDC 모터: 로봇의 구동부 

## 사용 기술
- C
- Python
- STM32CubeIDE
- OpenCV
- UART / TCP-IP 통신

## 주요 기능
- ArUco 마커 기반 위치 인식
- 주차 위치 탐색 및 경로 이동
- 차량 하부 진입 후 리프팅 동작
- 목적지까지 이동 후 주차 수행

## 담당 역할
- STM32 기반 모터 제어 및 센서 제어
- Jetson Nano와 MCU 간 통신 연동
- 전체 시스템 통합 및 동작 검증
